문제가 된 메시지

언젠가 부터 헥소를 이용해 글을 게시하면 저런 에러가 나왔지만 잘 실행 되엇기에(-_-) 무시하고 살았습니다. 그런데 최근에 Node.js ko저장소에서도 비슷하진 않지만 비슷한 에러 메시지가 나타나는걸 목격했고, 노드 버전을 올리니까 괜찮더라는 답글을 발견했습니다(나는 에러 메시지 따위는 읽지 않는다!).

노드 버전 업그레이드!

젤 먼저

를 실행 했습니다. 거침없는 업그레이드에 감탄하면서 이 글을 쓰기 시작 했습니다. 오늘은 너무 피곤했기 때문에 영어를 읽을 마음이 싹 달아났던 것이었던 거시었습니다…

뭐야 안되잖아?!

그래도 같은 메시지가 나타나서 결국 구글에 비슷한 메시지를 찾아 해결책을 달라고 요구했습니다(물론 검색창에…).

뭔가 내가 읽은 메시지와는 다른 모듈을 리빌드 하는것 같았지만 그냥 복붙했습니다(오늘은 힘드니까 ㅠㅠ).

역시 결과는 실패!

머야 이거 점점 복붙이 재밌어진다!

그래서 또 대충 검색을 막 때려봅니다. 그래서 얻어걸린 내용은

오 이번에는 뭔가 될것 같아!!!라는 느낌적인 느낌에 npm 을 업그레이드 할 수 있다는 광고 메시지까지!
그래서 한 번 광고를 따라 복붙합니다.

이쯤되면 이 사람이 왜 이러나 싶겠지만, 오늘은 아무 생각을 하기 싫었…ㅠㅠ

그래도 안되잖아! 에이… 정답을 찾자!

이제 슬슬 시간도 아깝고 해결하고 싶어집니다… 초조합니다. 떵줄이 탑니다. 왜냐면 시간은 소중하니까요.

짜잔 ~! 결국 마법의 명령어를 찾아냅니다! 역시 에러 메시지를 잘 읽지 않거니와 구글링을 잘 못하는 저같은 개발자는 혼꾸녕이 나야합니다.

참조 페이지에서… JLHwung님 감사합니다.
